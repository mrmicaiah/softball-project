# Softball Facts — by Kensey 🥎

A fun school project all about softball, made by Kensey.

## What's inside

- **`index.html`** — The main page with photos, fun facts, and the QUIZ button
- **`quiz.html`** — A Wheel of Fortune style quiz: Girls vs. Guys, 5 rounds!

## How the quiz works

1. Pick a letter — if it's in the answer, your team gets points (10 per letter)
2. Guess wrong? Other team's turn
3. Hit "SOLVE PUZZLE" anytime for a chance to win the whole round (+100 points)
4. Solving letter-by-letter gets you a +50 bonus
5. Highest score after 5 rounds wins!
